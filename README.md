# Connectify

A modern real-time social discovery and messaging platform where users connect, chat, and express themselves through mood-based profiles and interactive features.

🌟 Features
🔐 Authentication System
Secure login and registration with JWT authentication
🤝 Connection System
Send, accept, and manage connection requests
💬 Real-Time Chat
Seamless messaging experience with modern chat UI
👤 Interactive User Profiles
View profiles with bio, mood, and activity
🌈 Mood-Based Presence System
Users express their current mood (Happy, Chill, Focus, etc.)
🔍 User Discovery & Search
Find users using username search
🔔 Smart Notifications
Handle connection requests and activities in real-time
⚙️ Privacy Controls
Toggle online/offline visibility
❌ Disconnect Feature
Manage and remove connections easily
🛠️ Tech Stack

Frontend

React.js
CSS (Modern UI + Glassmorphism)

Backend

Node.js
Express.js

Database

MongoDB Atlas

Authentication
